Resume Analyser

The backend is running. Use the API endpoints below.

  • Upload resume: POST /api/resumes/upload (multipart form, param file)
  • Analyze for job: POST /api/resumes/analyze/{id} (path id, form param vacancyUrl or JSON body)
  • Tailor resume: POST /api/resumes/tailor/{id}
  • Approve & save: POST /api/resumes/approve/{id}
  • Get resume metadata: GET /api/resumes/{id}
  • H2 console (if enabled): /h2-console

Quick curl examples

Upload a file:

curl -v -F "file=@/path/to/resume.pdf" http://localhost:8080/api/resumes/upload

Get metadata (replace {id}):

curl http://localhost:8080/api/resumes/{id}

If you still see a Whitelabel Error Page when visiting /, refresh the browser cache (Shift+Refresh) or try an incognito window.
